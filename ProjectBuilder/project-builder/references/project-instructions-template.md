---

## Template Usage Notes

**Keep it focused:**
- One primary workflow per project
- Additional workflows can be added as needed

**Be specific:**
- "Write in active voice" beats "write well"
- "3-5 bullet points per section" beats "keep it brief"

**Include examples when possible:**
- Real examples from the user's domain
- Concrete illustrations of quality criteria

**Connect to sources:**
- Reference the manifest for dynamic data
- Reference uploaded files for static reference
