# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-16

### Added
- Initial release
- Dual-mode operation (Minimal ~200 words, Full ~1000 words)
- Auto-mode selection based on conversation complexity
- Antifragile structure (critical info first, survives truncation)
- Decision log with taxonomy (explicit/implicit/emergent)
- Handshake protocol for receiving agents
- Evolution tags ([G/C/P/K] or simplified)
- Python validator for artifact quality
