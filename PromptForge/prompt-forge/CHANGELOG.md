# Changelog

- Eight drill types (construction, diagnosis, refinement, adaptation, comparison, chain, stress test, teaching)
- Three session modes (Quick, Deep, Turbo)
- Persistent learning goals via forge-learning-goals.md
- Reference library: dimensions, techniques, drill patterns, learning goals template
