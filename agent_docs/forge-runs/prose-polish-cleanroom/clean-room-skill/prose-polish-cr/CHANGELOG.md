# Changelog

All notable changes to the prose-polish-cr skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-06-21

### Added
- **"Economy Must Not Cost Authority" (the vague-passage rule):** cutting a vague sentence shorter still leaves it vague; on a generic/hedged passage, sharpen-into-concrete (within the author's meaning) FIRST, else flag for the author `[specify: ...]`, tighten only as a last resort — never trade authority/voice for word count. (Round-1 forge fix: clean-room v0 over-cut the opinion essay into terse-but-flat and lost it to the no-skill baseline.)

## [0.1.0] - 2026-06-21

### Added
- Clean-room initial build (authored without consulting any existing prose skill): 6 dimensions (Clarity, Structure, Economy, Authority, Voice, Genre fit), top-down editing order (structure before line-level), meaning + voice preserved as constraints at every pass, genre-calibration table, before/after guidance.
