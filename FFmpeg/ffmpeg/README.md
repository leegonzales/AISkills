# ffmpeg

Complete FFmpeg toolkit for video/audio manipulation, analysis, and production.

## Installation

This is a Claude Code skill. Add it to your Claude Code configuration to use FFmpeg patterns during sessions.

### Prerequisites

- `ffmpeg` — `brew install ffmpeg`

## Usage

Invoke when working with video or audio files. Covers:

- **Probing** — inspect file metadata and stream info
- **Trim & concat** — cut and join video/audio segments
- **Convert** — transcode between formats (MP4, WebM, ProRes, MKV, etc.)
- **Scale & effects** — resize, crop, filters, overlays
- **Audio ops** — extract, mix, normalize, convert audio
- **Quality metrics** — PSNR, SSIM, VMAF analysis
- **Platform encoding** — YouTube, TikTok, Instagram, LinkedIn presets
- **HLS** — HTTP Live Streaming output
- **GIF** — animated GIF generation from video
- **Hardware acceleration** — GPU-accelerated encoding
- **Batch workflows** — process multiple files

## Triggers

- "ffmpeg", "transcode", "trim video", "convert video", "compress video"
- Any video/audio manipulation request
