# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-01

### Added
- Initial release
- 8-dimension scoring framework (Modularity, Composability, Clarity, Simplicity, Robustness, Data-Drivenness, Extensibility, AI-Readiness)
- 17 Unix rules reference with code smells and exemplar patterns
- SOLID principles mapped to scoring dimensions
- Architecture-type calibration (CLI, Library, Web App, Microservice, AI/ML)
- AI-Readiness dimension with 5 sub-scores
- Dimension gap diagnostics
- 3 operating modes: Quick Scan, Deep Review, AI-Readiness Audit
