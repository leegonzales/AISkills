This Go code implements a classic **Genetic Algorithm (GA)**, an optimization technique inspired by Charles Darwin's theory of natural selection. The goal is to "evolve" an optimal solution (in this case, a text `Prompt`) from an initial population of candidates over several generations.

The algorithm iteratively refines a population of individuals. In each generation, the fitness of every individual is evaluated, and the fittest individuals are stochastically selected to form a new generation through reproduction.

### Key Components in the Code

The three core operations of a genetic algorithm are clearly implemented in this code:

1.  **Selection**: This process determines which individuals from the current generation will be chosen as "parents" to breed the next generation. Fitter individuals have a higher chance of being selected.
    *   **In the code**: The `breedNextGeneration` function calls `e.population.SelectParents(e.config.SelectionPressure)`. The selection logic itself is abstracted away in the `PopulationManager`, but this is the clear entry point for the selection phase.

2.  **Crossover (or Recombination)**: This operation mimics biological reproduction. It takes two parent solutions and combines their "genetic material" to create one or more offspring. The idea is that the offspring might inherit the best characteristics of both parents.
    *   **In the code**: The `crossover(parent1, parent2 string)` function implements this. It performs a simple "single-point crossover" by selecting a random split point in the parent prompts and combining the first part of `parent1` with the second part of `parent2`. This is executed within `breedNextGeneration` based on the `CrossoverRate`.

3.  **Mutation**: This operation introduces random changes into an individual's genetic material. Mutation is crucial for maintaining genetic diversity within the population and preventing the algorithm from getting stuck in a local optimum.
    *   **In the code**: The `mutate(prompt string)` function handles this. It randomly selects a character in the prompt string and replaces it with another random character. This is executed within `breedNextGeneration` based on the `MutationRate`.
