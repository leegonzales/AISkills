# Changelog

All notable changes to SlideBuilder will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-01-19

### Added
- Initial skill release
- Talk Track v5 output format
- HTML presentation generation support
- Remotion video output support
- Image handling with USE/ADAPT/RECREATE/SKIP ratings
- Nano Banana prompt generation for RECREATE images
- Kokoro TTS integration (default voice)
- ElevenLabs v3 support for publish mode
- Semantic tag to slide mapping
- Speaker notes generation
- Section-based organization with colors
- Integration with essay-to-speech input format
- Complete reference documentation
