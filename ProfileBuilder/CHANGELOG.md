# Changelog

All notable changes to Profile Builder will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-24

### Added
- Initial release of Profile Builder skill
- Four-section structured interview process (Communication, Identity, Frameworks, Boundaries)
- Example profiles for 6 professional roles and archetypes
- Anti-pattern guidance for common mistakes
- Voice calibration to reflect authentic user voice
- Ready-to-paste output format for Claude Personal Preferences

### Structure
- `profile-builder/SKILL.md` - Core skill definition
- `profile-builder/references/examples.md` - Example profiles with analysis
- Distribution packaging support

### Documentation
- Complete README with installation instructions
- Example profiles with "Why it works" explanations
- Best practices for effective profiles
